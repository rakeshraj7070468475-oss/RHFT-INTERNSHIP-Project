# Safe Calculator

A secure and user-friendly calculator built using **C++**, **HTML**, **CSS**, and **JavaScript**.

This project performs basic arithmetic operations while handling invalid inputs and runtime errors safely using proper validation and exception handling.

---

## Features

✅ Addition  
✅ Subtraction  
✅ Multiplication  
✅ Division  

✅ Error Handling
- Division by zero
- Invalid operator
- Non-numeric input
- Empty input fields

✅ Continuous execution until user exits  
✅ Clean and responsive UI  
✅ Try-Catch Exception Handling

## How to Run

### C++ Version

1. Open terminal
2. Compile the program

```bash
g++ main.cpp -o calculator
```

3. Run the executable

```bash
./calculator
```

---

### Frontend Version

1. Open `index.html` in browser

OR

2. Use VS Code Live Server extension
