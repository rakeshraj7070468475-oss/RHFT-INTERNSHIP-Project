# Secure Login System in C++

A simple console-based secure login system built using C++.

This project demonstrates basic authentication logic, password validation, exception handling, and login attempt protection.

---

## Features

- Username and password authentication
- Password strength validation
- Maximum login attempt limit
- Exception handling using try-catch
- Console-based user interaction

---

## Technologies Used

- C++
- Object-Oriented Programming Concepts
- Exception Handling
- Standard Template Library (STL)

---

## How It Works

1. User enters username and password
2. System validates credentials
3. Incorrect attempts are tracked
4. System locks after multiple failed attempts
5. Password strength is checked before login
